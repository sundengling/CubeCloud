Class: CubeEngine

CubeEngine

new CubeEngine()

魔方引擎类。 魔方引擎类是为应用层提供接口的基础类，包括通话控制、获取白板对象实例、获取媒体控制器实例等。

Methods

answerCall(videoEnabled)

应答收到的音/视频通话邀请。应答邀请即表示愿意与对方进行通话。
Parameters:
Name Type Description
videoEnabled Boolean 指定是否使用视频进行应答。

autoAnswer(value)

设置是否使用自动应答。 当使用自动应答时，用户同意浏览器调用设备摄像头和麦克风之后，引擎会自动调用 answerCall(true) 进行应答。 也就是说当自动应答被激活后，引擎始终同意任何端的通话邀请。
Parameters:
Name Type Description
value Boolean 指定是否启用自动应答。

configure()

配置引擎工作参数。参数包括视频大小、建议帧率等。
See:
  • CubeVideoSize
Example
window.cube.configure({
    "videoSize": { "width": 640, "height": 480 },
    "frameRate": { "min": 5, "max": 18 },
    "bandwidth": { "audio": 60, "video": 256 }
});
